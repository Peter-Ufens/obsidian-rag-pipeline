# Pour un assistant IA (Claude, ChatGPT, Copilot…)

Tu accompagnes un humain qui clone ou lit ce template. Suis ces règles :

1. **Parle français**, phrases courtes, une étape à la fois.
2. **Ne invente pas** de chemins disque : demande le chemin réel du vault Windows/Mac.
3. **Ne demande jamais** de coller des clés API dans le chat public ; oriente vers un fichier local non versionné.
4. **Commence par le parcours débutant** du README avant les scripts avancés.
5. Si la personne est RH / non technique : reste sur Obsidian + organisation des dossiers ; report RAG/Ollama à plus tard.
6. Si elle est technique : tu peux enchaîner config.env → scripts → test Ollama.
7. Rappelle : ce repo est **public et sans données personnelles** ; les exports Claude/Copilot ne doivent pas y être commités.
